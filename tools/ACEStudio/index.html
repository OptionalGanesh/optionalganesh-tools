Voice Changer
Audio-to-audio timbre transfer · no text prompt · parameter-based
ParamsParameter Reference
▶
Pitch correction0–20% → natural feel  ·  20–40% → lightly corrected  ·  100% → hard autotune / T-Pain
Semitone offset0 = same  ·  +7 = fifth up  ·  -7 = fifth down  ·  ±3 = minor/major third  ·  ±12 = octave
Correct by scaleOn = quantizes to chosen scale (good for harmony shifts)  ·  Off = free pitch transfer
OffsetSmall value = slight timing humanization (use when stacking voices)

Right-click audio → AI Tools → Voice Changer. No text prompt needed.

RecipeQuick Harmony Stack
▶
Model 1 → offset 0, pitch correction 20%, no semitone shift → unison double Model 2 → offset +7, pitch correction 60%, correct by scale ON → fifth above Model 3 → offset -5, pitch correction 60%, correct by scale ON → third below Model 4 → small timing offset on each for human feel → Pan each model differently in the mixer → Lower all voice changer tracks vs lead vocal

Choose 3–5 different vocal models. Creates harmony stack without MIDI or lyric editing.
