SoundCloud RSS URL
Encuentra tu URL en SoundCloud → Settings → Content → RSS.
Formato: https://feeds.soundcloud.com/users/soundcloud:users:{TU_ID}/sounds.rss
Para encontrar tu ID numérico: abre soundcloud.com/tu-usuario en el navegador, click derecho → Ver código fuente → busca soundcloud:users:.
